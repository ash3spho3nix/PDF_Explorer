# MCP Bridge

Exposes PDF Inventory capabilities through the Model Context Protocol (MCP).

## Features
- Tools for search, document lookup, duplicates, similarity, statistics, etc.
- Resources for categories, statistics, folders, etc.
- Prompts for common tasks (weekly digest, summarize folder, etc.)
- Read‑only access; no changes to the core database.
- Compatible with Claude Desktop, VS Code, Cursor, and other MCP hosts.
